# gwasplot: Publication-Ready and Interactive Plots for GWAS Results

A ggplot2-based toolkit for visualising genome-wide association study
(GWAS) results. Provides linear Manhattan plots, per-chromosome plots
and circular (CMplot-style) plots with support for multiple concentric
trait rings, easy highlighting and labelling of top markers, interactive
versions via 'ggiraph', and companion tables of top hits via 'gt'.
