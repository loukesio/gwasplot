# Package index

## Plots

The three GWAS plot types.

- [`gwas_manhattan()`](https://loukesio.github.io/gwasplot/reference/gwas_manhattan.md)
  : Manhattan plot of GWAS results
- [`gwas_chromosome()`](https://loukesio.github.io/gwasplot/reference/gwas_chromosome.md)
  : Single-chromosome (regional) plot of GWAS results
- [`gwas_circular()`](https://loukesio.github.io/gwasplot/reference/gwas_circular.md)
  : Circular (CMplot-style) plot of GWAS results, with multiple trait
  rings

## Top markers and tables

Select, tabulate and highlight the strongest associations.

- [`highlight_top()`](https://loukesio.github.io/gwasplot/reference/highlight_top.md)
  : Specify which top markers to highlight
- [`gwas_top()`](https://loukesio.github.io/gwasplot/reference/gwas_top.md)
  : Extract the top GWAS markers as a tidy table
- [`gwas_table()`](https://loukesio.github.io/gwasplot/reference/gwas_table.md)
  : A formatted table of the top GWAS markers

## Large studies

Helpers for genome-wide data with millions of SNPs (see the “big_data”
argument on the plots).

- [`thin_gwas()`](https://loukesio.github.io/gwasplot/reference/thin_gwas.md)
  : Thin a large GWAS data frame for plotting

## Data contract and helpers

- [`validate_gwas()`](https://loukesio.github.io/gwasplot/reference/validate_gwas.md)
  : Validate and standardise a GWAS data frame
- [`order_chromosomes()`](https://loukesio.github.io/gwasplot/reference/order_chromosomes.md)
  : Order chromosome labels in natural genomic order
- [`simulate_gwas()`](https://loukesio.github.io/gwasplot/reference/simulate_gwas.md)
  : Simulate a GWAS summary-statistics data frame
- [`theme_gwasplot()`](https://loukesio.github.io/gwasplot/reference/theme_gwasplot.md)
  : A clean theme for gwasplot figures

## Example datasets

- [`gwas_example`](https://loukesio.github.io/gwasplot/reference/gwas_example.md)
  : Example single-trait GWAS results
- [`gwas_multi`](https://loukesio.github.io/gwasplot/reference/gwas_multi.md)
  : Example multi-trait GWAS results
